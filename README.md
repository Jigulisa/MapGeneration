##### Discription:
The program will provide the user with the opportunity to create flat map of some game world. The user will be able to choose
colors of the map, its size and file extension in case he want to save it. The program will start with cargo, i think.
The program will need only one form of the main menu. It'll contain checkboxes for file extensions, LineEdit for entering sizes
and colors and label for instructions. A database will be used for saving all generated pics, so the user will be able
to get any of generated (or saved if db will be at users computer) As the bd will contain pictures, its size can be
really big after some time of using program. At last, program will finish when the user will close it.
Just finishing, nothing spesial.
